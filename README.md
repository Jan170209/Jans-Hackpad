# Jans-Hackpad

A macropad with 6 keys and a knob to control volume etc. The little cutout at the top left is window to see the microcontroller.

# BOM
- 1x PCB
- 6x Cherry MX Switches
- 6x Keycaps
- 1x EC11 Rotary encoder
- 1x Seeed XIAO RP2040
- 4x M3x5x4 heatset inserts
- 4x M3x16mm screw
